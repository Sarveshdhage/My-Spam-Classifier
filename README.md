# 🚫 SMS & Email Spam Classifier

An end-to-end Machine Learning project to classify messages into **Spam** or **Ham** (Normal) using Natural Language Processing (NLP).

### 🚀 Project Highlights
- **Accuracy:** 98.39%
- **Algorithm:** Multinomial Naive Bayes
- **Tech Stack:** Python, Pandas, Scikit-learn, Jupyter Notebook

### 📊 Workflow
1. **Data Loading:** Used the SMS Spam Collection dataset from Kaggle.
2. **Text Preprocessing:** Used `CountVectorizer` to convert text into numerical features.
3. **Model Training:** Implemented a Scikit-learn Pipeline for seamless training.
4. **Evaluation:** Verified performance using a Confusion Matrix.

### 🛠️ How to Use
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Run the Jupyter Notebook or the Streamlit App.
